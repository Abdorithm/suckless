# suckless
My dwm, st, slstatus, &amp; dmenu configs. Currently using <code>Hack</code> and <code>Hack Nerd Font</code>. Running on Arch Linux, commands in slstatus may not work with a different distro and/or hardware and will need to be reconfigured accordingly.

Just clone the repo and run <code>sudo make clean install</code> inside each directory. You will need to install the mentioned fonts or change them to your liking in dwm's config file.
